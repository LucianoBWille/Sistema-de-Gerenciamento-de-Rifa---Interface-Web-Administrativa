### Ferramentas necessárias
1. Visual Studio Code
2. Extensão "Live Server" de Ritwick Dey

### como abrir o painel administrativo
1. Abra a pasta no VS Code (pode clicar com o botão direito na pasta e então em "Abrir com VS Code")
2. Clicar com o botão direito sobre o arquivo "index.html" e então em "Abrir com Live Server"

### como abrir o painel de busca do vencedor
1. Abra a pasta no VS Code (pode clicar com o botão direito na pasta e então em "Abrir com VS Code")
2. Clicar com o botão direito sobre o arquivo "winner.html" e então em "Abrir com Live Server"

OBS: para o sistema funcionar é necessário que a API também esteja inicializada
